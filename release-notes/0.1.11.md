# 0.1.11

## 新增

- 支持插件界面在中文和英文之间切换。

## 修复

- 修复编辑模式与阅读模式切换后，标注或评论偶发不可见、样式错乱的问题。
- 修复连续标注与评论背景在空格、行内代码及局部文字颜色组合下出现断裂的问题。
- 修复已有背景标注内局部设置文字颜色时范围扩大，并完善继承背景展示。
- 修复侧边栏与浮动工具栏在部分主题下的图标、菜单和颜色选择器显示问题。

## 优化

- 优化块级浮动菜单定位：下方可展开一半时支持滚动，空间不足时无重叠上浮。
- 统一正文与标题图标视觉尺寸，并整理工具栏间距和菜单分隔样式。
- 发布流程强制使用中文发布日志，并增加发布前校验。
