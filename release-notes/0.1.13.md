# 0.1.13

## 新增

- 新增阅读模式精准锚点映射，将渲染后的选区可靠还原到对应的 Markdown 源码范围
- 新增编辑模式锚点实时跟踪，正文编辑后标注位置与失联状态自动更新
- 新增编辑表格单元格标注渲染，在表格预览与活动单元格编辑间保持一致的定位和样式

## 修复

- 完善表格、链接与自动链接、任务列表、转义字符、内联代码、HTML 实体、代码围栏和跨块选区的源码定位
- 兼容预览块边界、空白源码行和不可见 DOM 字符，避免可见选区无法创建标注
- 加固重复文本和重叠标注处理，防止标注跨预览分段误匹配或嵌套样式覆盖错误
- 修复多文件锚点并发保存、旧状态回写、文件重命名与删除，以及外部编辑后的确定性恢复问题
- 修复预览分段元数据暂不可用时既有标注被清除的问题

## 优化

- 阅读模式复用源码、侧车数据和行偏移快照，并按受影响预览块局部重绘，降低大型文档的渲染开销
- 优化标注样式更新路径，连续背景跨普通文本、粗体和行内代码自然衔接，并移除悬停分块边框
- 优化侧栏标注的创建顺序、跨文件刷新和背景色辨识度
